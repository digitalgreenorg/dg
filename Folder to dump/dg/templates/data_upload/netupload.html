Static Data Upload

Static Data Uplaod lets you upload excel or csv sheets containing data collected from the field. This tool will add villages, groups and persons from file to database, and all those entries which could not be added will be notified to the user via email, which can also be downloaded from status page.

Instructions for uploading the file:

  • Data to be uploaded from the file should be from a single block

  • File should be in .csv/.xls/.xlsx form

  • Each excel file should contain only a single sheet of data

  • All fields mentioned in bold in sample sheet are necessary
